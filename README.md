# BBank 

Simple app to BBank API project

## Project setup
```
npm install
```

Create a `.env` with the variables like `.env.example`

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
